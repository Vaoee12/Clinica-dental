El Sistema Web de Gestión y Control Odontológico permite administrar de forma centralizada la información operativa y clínica de una red de consultorios odontológicos. Desde esta interfaz se pueden consultar y registrar pacientes, sucursales, odontólogos y consultorios, integrando el frontend con una API REST desarrollada en Flask y una base de datos PostgreSQL ejecutada en un entorno contenerizado con Docker.
